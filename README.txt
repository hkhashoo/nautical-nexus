Important note about fonts:
- The site uses the Roboto webfont via Google Fonts. When you open index.html directly using the file:// protocol,
  some browsers (Chrome, Edge) will block external network requests or treat them differently, which prevents the
  Roboto font from being fetched. This is why you may see the default system font instead.

How to run the site locally (recommended):
1) Open a terminal in the folder containing these files (where index.html is).
2) Run a simple HTTP server so the files are served over http:// instead of file://
   - Python 3:    python -m http.server 8000
   - Node (http-server): npx http-server -p 8000
3) Open your browser and navigate to:
   http://localhost:8000/
4) Fonts and other network resources (Google Fonts) will load correctly when served over HTTP.
